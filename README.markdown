gcal
====

A Command Line Interface to Google Calendar.

For more information, see http://blog.andrew-jones.com/tag/gcal/

Written in perl5.

INSTALL
-------

    curl -L http://cpanmin.us | perl - --sudo App::cpanminus
    cpanm App::gcal
